# Face to Face

Guía web para comparar **componentes y patrones de interfaz** entre **Apple Human Interface Guidelines (iOS)** y **Material Design 3 (Android / web)**. Incluye niveles de correspondencia, enlaces a documentación oficial, SDKs y kits de Figma.

Proyecto de apoyo al curso **Diseño y desarrollo de UI para apps** de [Shift+R](https://www.shiftr.pro/).

## Tres versiones visuales

El sitio ofrece un conmutador de tema en la cabecera:

| Tema | Descripción |
|------|-------------|
| **Default** | Estilo base claro, tipografía Poppins, sidebar y contenido equilibrados. |
| **Material** | Apariencia **Material Design 3**: colores dinámicos, superficies M3, Top App Bar y componentes alineados con la guía de Google. |
| **iOS** | Estilo **MacTahoe / iOS**: vidrio, squircles, acento azul tipo sistema y fondo tipo wallpaper en escritorio (móvil con fondo plano legible). |

Los tres comparten la misma información y navegación; solo cambia la capa visual.

## Cómo usarlo

- Abre **`index.html`** en un navegador moderno, o
- Sirve la carpeta con cualquier servidor estático (por ejemplo `npx serve .`).

No hace falta build: **HTML + CSS + JavaScript** en vanilla.

## Contenido principal

- Listado de pares iOS ↔ Material con **alta / media / baja** correspondencia.
- Fichas de componente con descripción, enlaces HIG / M3 y referencias.
- Página **Acerca de** con contexto del proyecto y kits oficiales en Figma.

## Créditos
